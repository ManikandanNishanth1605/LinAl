# Linear Algebra Codes
This repository contains algorithms and matrix operations I learnt from linear algebra courses and books. The codes might not be optimal but I am looking to improve them. These codes are for educational purposes only. Any suggestions and improvements are welcome.

## How to run the codes
It is assumed that `cmake` and `g++` are installed in the machine. Please install them before running the codes.
1. Clone the repository using `git clone https://github.com/ManikandanNishanth1605/LinAl.git`
2. Go to the directory of the repository using `cd LinAl`
3. Create a build directory and go into the directory using `mkdir build && cd build`
4. Compile the code using `cmake .. && make`
5. Run the executable using `./runit`.
